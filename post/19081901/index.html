nssm生成系统服务

Sun, Aug 25, 2019

nssm生成服务

nssm可以将Nodejs项目生成Windows系统服务。
下载对应nssm版本（32/64）放到将生成服务的项目文件夹中
在项目文件夹中打开shell
输入 >nssm install [项目名称] 弹出nssm界面
>pash: 选择项目.exe
>Startup direcotry: 默认选择项目目录
>Arguments: 启动参数
点击 Install service 生成服务
>net start [项目名称] 启动服务 也可以右键计算机管理–服务和应用程序–服务
nssm常用命令
nssm install servername //创建服务
nssm start servername //启动服务
nssm stop servername //暂停服务
nssm restart servername //重新启动服务
nssm remove servername //删除创建的服务
