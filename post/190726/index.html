安装程序脚本 静默安装

Fri, Jul 26, 2019

碰到多台电脑需要安装相同驱动与软件时，可选择将所有软件拷贝到同一文件夹中。

建立txt文件

内容为

@echo off

start /wait 软件名.exe

start /wait 软件名.exe

pause

rem

…

等

完成后，文件另存为bat（或直接更改扩展名）

统一copy到电脑后运行 bat脚本便可

静默安装也可称为无人值守安装，
指的是安装时无需任何用户干预，
直接按默认设置安装，
我们在安装一些程序的时候一直要点击下一步，
其实想要免除这些麻烦，直接静默安装就可以了。

常用软件静默安装参数

7-ZIP.exe /S

WinRAR.exe /S

迅雷 /S

金山词霸 /S

百度云 /S

QQ /S

微信 /S

有道词典 /S

360驱动大师（装机版） /q

驱动精灵（装机版） -a

万能驱动 /fa /c

IT天空VC++ /Q

IT天空.NET /Q

微软VC++ /S 或者 /Q

微软.NET /S 或者 /Q

Google拼音 /S

搜狗输入法 /S

酷狗音乐 /S

驱动精灵 /S

CPU-Z /S

小马激活工具 静默参数 /a /acer /random /protect

举例静默安装7zip

新建txt文本，内容如下

@echo off

start /wait 7Zip.exe /S

pause

rem

另存为扩展名bat。右键运行
