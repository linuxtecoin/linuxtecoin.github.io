Hugo 搭建记录

前方操作请参照

Hugo 中文文档

https://www.gohugo.org/

很详细但最后

cd public

git init

git remote add origin

git add -A

git commit -m “first commit”

git push -u origin master

看着就懵了。 如果没有git基础的话

从第一步开始。

先按照文档搭建，直到本地预览成功

然后

hugo - -theme=主题名 - -baseUrl=‘仓库名即Github博客名’ –buildDrafts

这一步会生成 publice 文件夹

进入文件夹

cd public

剩下的严格按照文档操作即可

git init 　　　 #创建仓库

git add . 　　 #需要提交的代码从工作区添加到暂存区

git commit -m ‘这里是提交的备注’

git remote add origin 仓库完整地址从http://github.com开始直到.io并后面跟.git

注意第二次及以后推送会产生

fatal: remote origin already exists.

需要先删除之前的 origin 可以继续操作

删除命令

git remote rm origin

然后继续上传

最后，推送到github

git push

等待推送成功，如果推送失败则使用

git push -u origin master

会提示登录用户名与密码，登陆后会推送成功

现在打开你的github.io便可以看到你的Blog。
